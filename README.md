# Authentication System using MERN Stack with JWT

## Project Overview

This is a simple **Authentication System** built using the **MERN stack** with JWT-based authentication.  
It supports **role-based login** (Admin/User) and a basic **profile page** to test authentication functionality.

---

## Tech Stack

- **Frontend:** React + TailwindCSS
- **Backend:** Node.js + Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT (JSON Web Tokens)

---

## Features

- User/Administrator login with JWT authentication
- Role-based access control (Admin/User)
- Profile page to test protected routes
- Simple backend API with REST endpoints
- Ready for deployment on Vercel/Heroku

---

## Prerequisites

- Node.js (v18+ recommended)
- MongoDB database (Atlas or local)
- npm or yarn package manager

---

## Project Setup

### Backend

```bash
cd backend
npm install
npm run dev
```
